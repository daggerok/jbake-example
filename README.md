# jbake [![Build Status](https://travis-ci.org/daggerok/jbake-example.svg?branch=master)](https://travis-ci.org/daggerok/jbake-example)

```bash
# create
mkdir jbake-example
cd jbake-example
jbake -i

# build
jbake -b

# serve with jetty
jbake -s
open http://localhost:8820/
```
